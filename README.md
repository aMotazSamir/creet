# creet

أداة بسيطة لتحديد اختصار تفضّله وعرض قائمة اختصارات ويندوز قابلة للتعديل.

## الفكرة
- تختار اختصار عام (Hotkey).
- عند الضغط عليه تظهر لك نافذة تحتوي على الاختصارات التي تحددها.
- تستطيع تعديل الاختصارات والوصف من نافذة الإعدادات.

## التشغيل
1. ثبّت المتطلبات:
   ```bash
   pip install keyboard
   ```
2. شغّل الأداة:
   ```bash
   python shortcut_tool.py
   ```

> ملاحظة: مكتبة `keyboard` تحتاج صلاحيات مناسبة على ويندوز لتسجيل الاختصار العام.

## التخصيص
- يمكنك تعديل الاختصار العام وقائمة الاختصارات من زر **Settings**.
- يتم حفظ الإعدادات في `config.json`.
